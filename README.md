🌱 Crop Disease Detection using AI

This project focuses on building an AI-powered system for early detection of crop diseases using leaf images. It leverages deep learning (CNNs) with transfer learning and ensemble techniques to achieve accurate and robust classification across multiple crop types.

🚀 Features
Multi-class crop disease classification
Transfer learning (EfficientNet, ResNet, MobileNet)
Image preprocessing & data augmentation pipeline
Ensemble model for improved accuracy
Explainable AI using Grad-CAM
Designed for web/mobile deployment
🛠️ Tech Stack
Python
TensorFlow / Keras
OpenCV
Scikit-learn
📂 Project Status

🚧 Work in progress — core model and pipeline under development.

🎯 Goal

To develop a practical, scalable solution that helps farmers detect crop diseases early and take timely action.
